# STA 161 – Workplace Safety Incident Analysis Dashboard

## Project Overview

This project was developed as part of **STA 161** and focuses on analyzing workplace safety incidents using Microsoft Excel. The workbook contains raw incident data, pivot tables, calculations, and an interactive dashboard designed to provide insights into workplace injuries, incident costs, lost workdays, and demographic trends.

The goal of the project is to transform operational safety data into meaningful business intelligence that can support decision-making and improve workplace safety management.

## Dataset Description

The dataset contains **514 workplace incident records** collected between **2020 and 2022**.

### Variables Included

| Variable | Description |
|-----------|-------------|
| Date | Date of incident |
| Injury Location | Body part affected |
| Gender | Employee gender |
| Age Group | Employee age category |
| Incident Type | Type of workplace incident |
| Days Lost | Number of workdays lost |
| Plant | Facility/plant location |
| Report Type | Incident severity classification |
| Shift | Work shift during incident |
| Department | Department where incident occurred |
| Incident Cost | Financial cost of the incident |
| Weekday | Day of week |
| Month | Month of occurrence |
| Year | Year of occurrence |

## Dashboard Features

The dashboard provides interactive visualizations and summaries of workplace incidents, including:

### 1. Incident Cost Analysis
- Total incident costs by incident type
- Cost trends across years (2020–2022)
- Identification of high-cost incident categories

### 2. Lost Workday Analysis
- Days lost across departments
- Comparison across shifts
- Departmental safety performance insights

### 3. Demographic Analysis
- Incident distribution by age group
- Gender-based incident analysis
- Workforce risk profiling

### 4. Report Type & Injury Location Analysis
- Incident counts by report classification
- Injury location distribution
- Comparison across report types

### 5. Lookup & Calculations
- Excel VLOOKUP-based search functionality
- Dynamic retrieval of incident cost data
- User-friendly incident type selection

## Key Analysis Questions

The project helps answer questions such as:

- Which incident types generate the highest costs?
- Which departments experience the most lost workdays?

- How do injury locations vary across report types?
- What safety trends emerge over the three-year period?

## Sample Findings

Based on the analysis:

- **Burn**, **Equipment**, and **Crush & Pinch** incidents contribute significantly to overall costs.
- The workforce is predominantly represented by the **25–34** age group in reported incidents.
- Male employees account for a higher proportion of recorded incidents.
- Lost workdays vary noticeably across departments and shifts.
